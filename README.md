An attempt in spamming surveynuts of my school confession form.

## Usage
### Cloning
You have to clone with submodules.
`git clone --recurse-submodules https://github.com/dzhoz0/spamcfs-2024`
### Build libs
`make libs`
### Build program
`make build`
### Do everything
`make`
### Run
`./bin/spamcfs`
